## ✅ 문제 1: 중복 제거
### 🔍 풀이
```python
nums = [1, 2, 2, 3, 4, 4, 5]
unique = list(set(nums))
print("중복 제거:", unique)
```

## ✅ 문제 2: 숫자 뒤집기
### 🔍 풀이
```python
n = int(input("정수 입력: "))
reversed_n = int(str(n)[::-1])
print("뒤집은 수:", reversed_n)
```
