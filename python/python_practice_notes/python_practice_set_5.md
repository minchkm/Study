# 🐍 Python 연습문제 세트 5

## ✅ 문제 1: 소문자만 추출
### 🔍 풀이
```python
text = "Hello PyTHon"
only_lower = ''.join([c for c in text if c.islower()])
print("소문자만:", only_lower)
```

## ✅ 문제 2: 1부터 100까지의 합
### 🔍 풀이

## ✅ 문제 3: 리스트 최대/최소 차이
### 🔍 풀이
```python
lst = [10, 20, 5, 8, 15]
print("최대 - 최소 =", max(lst) - min(lst))
```
