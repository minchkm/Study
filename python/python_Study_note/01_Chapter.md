# Hello, Python!

## 파이썬이란?
- **쉽고 간결한 문법**을 가진 프로그래밍 언어로, 초보자부터 전문가까지 널리 사용됨
- **다양한 분야**에서 활용되며, 특히 웹 개발, 데이터 분석, 인공지능, 자동화 등에서 강력한 기능을 발휘
- **크로스 플랫폼**: 운영체제에 구애받지 않고 다양한 환경에서 실행 가능
- **풍부한 라이브러리**: NumPy, Pandas, TensorFlow, Flask 등 다양한 라이브러리로 여러 가지 작업을 쉽게 처리할 수 있음
- **인터프리터 언어**: 코드 작성 후 바로 실행이 가능해, 실시간으로 결과를 확인하며 개발할 수 있음

## 파이썬의 특징
- **가독성 좋은 코드**: 코드가 명확하고 직관적이어서 다른 사람과 협업 시에도 유리
- **동적 타이핑**: 변수의 타입을 명시할 필요 없이 값에 따라 자동으로 타입이 결정됨
- **고급 데이터 구조**: 리스트, 튜플, 딕셔너리 등 강력한 내장 데이터 구조 제공
- **대화형 개발 환경**: Python shell이나 Jupyter Notebook 등을 통해 실시간으로 코드를 실행하고 테스트 가능

## 첫 번째 파이썬 코드

```python
print("Hello, Python!")
```
이 코드는 화면에 "Hello, Python!"을 출력합니다. print()는 파이썬에서 기본적으로 제공하는 함수로, 주어진 값을 콘솔에 출력하는 역할을 합니다.

파이썬 실행 방법
터미널에서 실행: python 파일명.py 명령어를 통해 실행

예: python hello.py

Jupyter Notebook: 데이터 분석 및 실험을 실시간으로 진행할 수 있는 웹 기반 환경

IDLE: 파이썬 공식 IDE로, 기본적인 코드 작성 및 실행 환경 제공

VSCode, PyCharm: 고급 개발 환경으로, 코드 자동 완성, 디버깅, 버전 관리 등 다양한 기능을 제공

파이썬으로 할 수 있는 것들
웹 개발: Django, Flask 같은 웹 프레임워크를 사용하여 웹 애플리케이션 구축

데이터 분석: Pandas, NumPy 등 라이브러리로 데이터 처리 및 분석

인공지능(AI): TensorFlow, PyTorch와 같은 라이브러리를 사용해 머신러닝 모델을 만들고 훈련

스크립트 작성: 반복적인 작업을 자동화하는 스크립트 작성

게임 개발: Pygame을 사용해 간단한 게임 개발 가능

파이썬의 장점
쉽고 간단한 문법: 다른 언어에 비해 배우기 쉽고 직관적인 문법을 제공

활발한 커뮤니티: 다양한 사용자와 개발자들이 활발하게 정보를 공유하는 커뮤니티가 존재

무료 오픈소스: 누구나 자유롭게 사용할 수 있고, 수정하거나 배포 가능
