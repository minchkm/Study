# 💡 자바스크립트 연습문제 모음

## 1. 짝수/홀수 판별기
사용자로부터 숫자를 입력받아 짝수인지 홀수인지 출력하세요.

## 2. 구구단 출력기
2~9단까지 구구단을 출력하세요 (중첩 for문 활용)

## 3. 배열 평균 구하기
숫자로 이루어진 배열의 평균을 구하는 함수를 작성하세요.

## 4. 팩토리얼 함수 만들기
숫자 n을 받아 n!을 반환하는 함수를 만들어보세요 (재귀 함수 활용)

## 5. 문자열 뒤집기
입력받은 문자열을 뒤집어서 반환하는 함수를 만들어보세요.

## 6. 최대값 찾기
배열에서 가장 큰 값을 찾아서 반환하는 함수를 만들어보세요.

## 7. 간단한 계산기
두 수와 연산자(+,-,*,/)를 입력받아 계산 결과를 출력하세요.

## 8. DOM 조작 연습
- 버튼을 클릭하면 배경색이 바뀌도록 하세요.
- 리스트에 새로운 항목을 추가하는 버튼을 만들어보세요.
